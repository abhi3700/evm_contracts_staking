## Features
* People can stake PREZRV tokens.
* Access NFT marketplace and services with staking tiers:
	- `$500` to unlock NFTs, 
	- `$1000` for NFT services, and 
	- `$1500` for DAO. 
* The minimum staking duration is `1 month` in order to get access to PREZRV services.
